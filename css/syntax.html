Syntax
Syntax är grunden i css. Det är med syntax men “säger” till sidan vad som ska hända till det du vill ändra. Så om man vill ändra på Vi säger vi vill ändra på nästa rad. Så ska vi först ge raden en ID eller class eller bara en tagg (mer om det i html delen). I css så definierar vi vad vi vill ändra med antingen tagen som t.ex p, artical, div mm. Om man har en id på texten så ska man göra en # sen skrivar man vilket id man gav den T.ex #text. Det är lika med class fast istället så ska man skriva en punkt innan stället för en #. t.ex .text. Det som behövs sen är att man “säger” vad det är man vill göra. Det gör man genom att inom { } så skriver man t.ex color om man ändra färg t.ex #text {color:red;}. :red; är värdet man ger det kan vara allt från storlekar till vara true eller false.
Exempel

#td{                   header{
width: 50%;    height: 10%;
height: 100%;  font-size: 3em;
padding: 0px;  padding: 0px;
margin: 0px;   margin: 0px;
}                  text-align: center;
}
